# Documentation

Learn how to use Courier to design and deliver notifications across multiple channels (email, SMS, mobile push, in-app) and integrate rich inbox experiences into your web and mobile apps.

## Getting Started

You can get started building with Courier for free. Learn [what Courier is](./getting-started/what-is-courier), what the [key concepts](./getting-started/courier-concepts) are how to send your first notification using your programming language and framework of choice.

## Tutorials

Our tutorials help developers learn basic Courier platform concepts, everything from [adding providers](./tutorials/how-to-add-an-integration-to-a-channel) to [sending notifications to a list](./tutorials/how-to-send-to-a-list-or-list-pattern-using-wildcarding).

## REST API

The [Courier API](/reference) is organized around REST. Our API has predictable resource-oriented URLs, accepts JSON-encoded request bodies, returns JSON-encoded responses, and uses standard HTTP response codes, authentication, and verbs.

## SDK Libraries

### Server SDK Libraries

You can access our REST API and integrate Courier into your application using one of our [server SDK](/sdk-libraries/server-sdks) libraries.

<TabItem value="ruby" label="courier-ruby">

```ruby
require "trycourier"

begin
  client = Courier::Client.new "your-auth-token" # or set via COURIER_AUTH_TOKEN env var
  res = client.send_message({
    "message" => {
      "to" => {
        "email" => "marty_mcfly@email.com",
        "data" => {
          "name" => "Marty McFly"
        }
      }
      "content" => {
        "title" => "hello {{name}}",
        "body" => "Welcome to Courier!"
      },
      "data" => {
        "name" => "Ruby"
      }
    }
  })
  puts res.code # the HTTP response code
  puts res.request_id # if the code is 202, this will be the Courier request ID for this message
rescue Courier::CourierAPIError => re #error sent from from the API
  puts re.message
end
```

</TabItem>

<TabItem value="go" label="courier-go">

```go
package main

import (
	"context"
	"fmt"
	"github.com/trycourier/courier-go/v2"
)

func main() {
	client := courier.CourierClient("<YOUR_AUTH_TOKEN>", nil)
	message := courier.SendMessageRequestBody{
		  "template": "<COURIER_TEMPLATE>",
			"to": map[string]string{
				"email": "marty_mcfly@email.com",
        "data": map[string]string{
          "name": "Marty McFly"
        }
		}
	}

	reqID, err := client.SendMessage(context.Background(), message)
	if err != nil {
		panic(err)
	}

	fmt.Println(reqID)
}
```
