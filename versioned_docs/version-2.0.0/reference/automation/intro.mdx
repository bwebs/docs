# Automations API

The Automations API provides endpoints to:

- Invoke an existing automation template
- Invoke an adhoc automation (An automation supplied in the call via JSON)
- Get information about an automation run, the "Automation Run Object:

To learn more about Automations, including examples, instructions on how to use the designer, and information about what Automations
can do visit the [automations guide](/platform/automations).
