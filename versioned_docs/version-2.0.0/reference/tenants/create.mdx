# Create or Replace a Tenant

**URL:** `https://api.courier.com/tenants/:tenant_id`

**Method:** `PUT`

**Path Parameters:**

<Params>
  <Param name="tenant_id" type="string" required>
    Id of the tenant to be returned.
  </Param>
</Params>

**Body:**

<TenantRequest />

**Response Body**

A copy of the created account.

<TenantResponse />

**Example:**

```js
// PUT https://api.courier.com/tenants/ACME

// BODY
{
  name: "ACME Inc",
  brand_id: "acme_brand",
  properties: {
    ceo: "Person",
  },
  default_preferences: {
    items: [{
      id: "my-topic",
      status: "OPTED_IN"
    }]
  },
  user_profile: {
    locale: "en-US",
    "slack": {
      "access_token": "xoxb-bunch-o-letters",
    }
  }
}

// Response Body
{
  id: "ACME",
  name: "ACME Inc",
  brand_id: "acme_brand",
  properties: {
    ceo: "Person",
  },
  default_preferences: {
    items: [{
      id: "my-topic",
      status: "OPTED_IN",
      type: "subscription_topic"
    }]
  },
  user_profile: {
    locale: "en-US",
    "slack": {
      "access_token": "xoxb-bunch-o-letters",
    }
  }
}
```
