# Get an Account

**URL:** `https://api.courier.com/accounts/:account_id`

**Method:** `GET`

**Path Parameters:**

<Params>
  <Param name="account_id" type="string" required>
    Id of the account to be returned.
  </Param>
</Params>

**Response Body:**

<AccountResponse />

**Example:**

```js
// GET https://api.courier.com/accounts/ACME

// RESPONSE BODY
{
  id: "ACME",
  name: "ACME Inc",
  properties: {
    ceo: "Person",
  },
  default_preferences: {
    items: [{
      status: "OPTED_IN",
      id: "my-topic",
      type: "subscription_topic"
    }]
  },
  user_profile: {
    locale: "en-US",
    "slack": {
      "access_token": "xoxb-bunch-o-letters",
    }
  }
}
```
