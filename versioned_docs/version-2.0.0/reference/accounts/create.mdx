# Create or Replace an Account

**URL:** `https://api.courier.com/accounts/:account_id`

**Method:** `PUT`

**Path Parameters:**

<Params>
  <Param name="account_id" type="string" required>
    Id of the account to be returned.
  </Param>
</Params>

**Body:**

<AccountRequest />

**Response Body**

A copy of the created account.

<AccountResponse />

**Example:**

```js
// PUT https://api.courier.com/accounts/ACME

// BODY
{
  name: "ACME Inc",
  brand_id: "acme_brand",
  properties: {
    ceo: "Person",
  },
  default_preferences: {
    items: [{
      id: "my-topic",
      status: "OPTED_IN"
    }]
  },
  user_profile: {
    locale: "en-US",
    "slack": {
      "access_token": "xoxb-bunch-o-letters",
    }
  }
}

// Response Body
{
  id: "ACME",
  name: "ACME Inc",
  brand_id: "acme_brand",
  properties: {
    ceo: "Person",
  },
  default_preferences: {
    items: [{
      id: "my-topic",
      status: "OPTED_IN",
      type: "subscription_topic"
    }]
  },
  user_profile: {
    locale: "en-US",
    "slack": {
      "access_token": "xoxb-bunch-o-letters",
    }
  }
}
```
