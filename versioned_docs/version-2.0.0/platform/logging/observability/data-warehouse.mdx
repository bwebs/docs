# Data Warehouse

Courier supports sending observability events to Azure Data Lake Storage, Azure Synapse, BigQuery, Databricks, MySQL, OneLake, Panoply, Periscope, PostgreSQL, Redshift, S3 Data Lake, Snowflake, and SQL Server empowering our customers to have complete visibility into their notification usage and perform complex analysis alongside the rest of their data.

:::info INFO
This feature is in Private Preview and is available to [Enterprise-tier](https://www.courier.com/pricing/) customers. Please contact your account team for access to this feature, and they will send you setup instructions for your data warehouse provider.
:::

### Available Data
Please see the complete list of [observability events](/docs/platform/logging/observability/intro) for events available to be sent to your data warehouse. The expected latency for this feature while it's in Private Preview is 12-24 hours.

### Snowflake Data Share
If you are a Snowflake customer, Courier supports [Snowflake data sharing](https://docs.snowflake.com/en/user-guide/data-sharing-intro) or directing events to your Snowflake tables.
